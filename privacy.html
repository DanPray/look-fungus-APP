隱私權政策

最近更新日期：2026年4月28日

「蕈菇調查紀錄 (Look@Fungus)」（以下簡稱「本應用程式」）非常重視您的隱私權。本政策旨在說明我們如何處理您的個人資料。

1. 資料收集與使用

本應用程式主要用於記錄野外真菌調查數據。當您使用調查功能時，我們會收集：

  • 調查數據：包含經緯度、日期、真菌特徵描述。
  • 影像資料：經您授權後，我們會存取相機及相簿以記錄調查照片。

2. 資料儲存與傳輸

本應用程式透過加密的 REST API 將您的調查數據上傳至伺服器。除非經您同意或法律要求，我們不會將您的資料分享給第三方。

3. 裝置權限說明

  • 相機/相簿：僅用於拍攝與儲存真菌觀測照片。
  • 位置資訊：僅用於標記調查地點的地理座標。

4. 聯絡我們

若對本隱私權政策有任何疑問，請透過技術支援網址與開發者聯繫。
